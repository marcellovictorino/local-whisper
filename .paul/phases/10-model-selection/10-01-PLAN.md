<objective>
## Goal
Switch default model to `mlx-community/distil-whisper-large-v3` (~2× faster than turbo at runtime, same ~1.5 GB download, <1% WER on English). Let users override via `[whisper] model` in config.toml. Document `mlx-community/whisper-large-v3-turbo` as the switch for multilingual support or higher accuracy.

## Purpose
Transcription latency is the primary UX friction. `distil-whisper` cuts it ~6×. One config
key + benchmark establishes baseline for further speed improvements in v0.6.

## Output
- `transcribe.get_model()` reads config, falls back to default distil-whisper
- `App` and `__main__` pass the resolved model to `warm_up()` and `run()`
- First-run size hint is dynamic per model
- `benchmark.py` module: warm-up time + transcription time over 3 runs of 30s synthetic audio
- `--benchmark` CLI flag + `just benchmark` recipe for repeatable baseline measurement
- 4+ new tests covering config read and default fallback
</objective>

    **benchmark.py (new file):**
    ```python
    """Benchmark transcription latency for comparison across models and backends."""
    from __future__ import annotations

    import json
    import sys
    import time
    from pathlib import Path

    import numpy as np


    _SAMPLE_RATE = 16_000
    _DURATION_S = 30


    def run(model: str, runs: int = 3) -> dict:
        """Benchmark warm-up and transcription latency.

        Args:
            model: HuggingFace model ID to benchmark.
            runs: Number of transcription runs (default 3).

        Returns:
            Dict with keys: model, warmup_s, runs, mean_s, min_s, max_s, times_s.
        """
        from local_whisper import transcribe

        audio = np.zeros(_SAMPLE_RATE * _DURATION_S, dtype="float32")

        # Warm-up
        t0 = time.perf_counter()
        transcribe.warm_up(model)
        warmup_s = time.perf_counter() - t0

        # Transcription runs
        times: list[float] = []
        for _ in range(runs):
            t0 = time.perf_counter()
            transcribe.run(audio, model=model)
            times.append(time.perf_counter() - t0)

        return {
            "model": model,
            "audio_duration_s": _DURATION_S,
            "warmup_s": round(warmup_s, 3),
            "runs": runs,
            "mean_s": round(sum(times) / len(times), 3),
            "min_s": round(min(times), 3),
            "max_s": round(max(times), 3),
            "times_s": [round(t, 3) for t in times],
        }
    ```
