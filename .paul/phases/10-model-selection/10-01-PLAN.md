<objective>
## Goal
Let users pick the Whisper model via `[whisper] model` in config.toml. Default stays
`mlx-community/whisper-large-v3-turbo`. Document `mlx-community/distil-whisper-large-v3`
as the fast alternative (~600 MB, ~6× faster, minor accuracy tradeoff).

## Purpose
Transcription latency is the primary UX friction. `distil-whisper` cuts it ~6×. One config
key unlocks this for users who prefer speed over accuracy — no code change required.

## Output
- `transcribe.get_model()` reads config, falls back to default
- `App` and `__main__` pass the resolved model to `warm_up()` and `run()`
- First-run size hint is dynamic per model
- 4+ new tests covering config read and default fallback
</objective>
