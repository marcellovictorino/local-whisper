<objective>
## Goal
Show a native macOS frosted-glass pill overlay ("⏺  Recording...") when Right Command is held. Fades in on keypress, fades out after paste. No dock icon. Uses PyObjC (already installed — pynput dependency).

## Purpose
Provides real feedback that the app registered the keypress and is recording. Without it users press the key twice wondering if it worked. The native AppKit look matches the macOS aesthetic (think Wispr Flow).

## Output
- `overlay.py` — `RecordingOverlay` (NSPanel + NSVisualEffectView HUD, fade animation, queue-driven, no dock icon)
- `app.py` updated — accepts optional overlay, `start()`/`stop()` methods, show/hide hooks
- `__main__.py` updated — `overlay.run()` owns main thread (AppKit event loop); `app.start()` runs listener in daemon thread

## Deferred (v0.2)
Sound wave bars that animate to audio amplitude — requires piping RMS from `record_until_event()` callback to a Core Animation layer. Architecture note added to SUMMARY for easy pickup.
</objective>

    ```python
    import queue
    ```

    `applicationDidFinishLaunching_` method:
    - Call `self._build_panel()`
    - Schedule NSTimer repeating every 0.05s calling `pollQueue_:`

    `pollQueue_:` method (NSTimer callback):
    - Drain self._queue with get_nowait() in a loop
    - "show" → `_fade_in()`
    - "hide" → `_fade_out()`
    - "quit" → `NSApplication.sharedApplication().terminate_(None)`
    - Catch `queue.Empty` to exit loop

    `_build_panel` method:
    - Get screen dimensions: `NSScreen.mainScreen().frame()`
    - Window size: `w=200, h=44`
    - Position: bottom-right `(sw - w - 20, 60)` — `y=60` from bottom (macOS coords are bottom-up)
    - Create `NSPanel.alloc().initWithContentRect_styleMask_backing_defer_(NSMakeRect(x, y, w, h), NSWindowStyleMaskBorderless, 2, False)`
    - `panel.setLevel_(NSFloatingWindowLevel)`
    - `panel.setOpaque_(False)`
    - `panel.setBackgroundColor_(NSColor.clearColor())`
    - `panel.setAlphaValue_(0.0)` — hidden initially
    - `panel.setIgnoresMouseEvents_(True)` — click-through
    - `panel.setCollectionBehavior_(NSWindowCollectionBehaviorCanJoinAllSpaces | NSWindowCollectionBehaviorStationary)`
    - `panel.setHasShadow_(True)`

    Frosted glass content view:
    - `effect = NSVisualEffectView.alloc().initWithFrame_(NSMakeRect(0, 0, w, h))`
    - `effect.setMaterial_(15)` — `NSVisualEffectMaterialHUDWindow` (dark frosted HUD)
    - `effect.setBlendingMode_(0)` — `NSVisualEffectBlendingModeBehindWindow`
    - `effect.setState_(1)` — `NSVisualEffectStateActive`
    - `effect.setWantsLayer_(True)`
    - `effect.layer().setCornerRadius_(14.0)`
    - `effect.layer().setMasksToBounds_(True)`
    - `panel.setContentView_(effect)`

    Label:
    - `label = NSTextField.labelWithString_("⏺  Recording...")`
    - `label.setFrame_(NSMakeRect(0, 0, w, h))`
    - `label.setAlignment_(2)` — center

    `_fade_in` method:

    ```python
    NSAnimationContext.runAnimationGroup_(

    ```

    `_fade_out` method:
    ```python

        lambda ctx: (ctx.setDuration_(0.20), self._panel.animator().setAlphaValue_(0.0))
    )
    ```

    **RecordingOverlay class:**
    ```python
    class RecordingOverlay:
        def __init__(self) -> None:
            self._queue: queue.Queue[str] = queue.Queue()

        def show(self) -> None:
            """Signal overlay to fade in. Thread-safe."""
            self._queue.put("show")

        def hide(self) -> None:
            """Signal overlay to fade out. Thread-safe."""
            self._queue.put("hide")

        def quit(self) -> None:
            """Signal AppKit event loop to exit. Thread-safe."""
            self._queue.put("quit")

        def run(self) -> None:
            """Start AppKit event loop. MUST be called from main thread. Blocks until quit()."""
            app = NSApplication.sharedApplication()
            app.setActivationPolicy_(2)  # NSApplicationActivationPolicyAccessory — no dock icon

            delegate = _AppDelegate.alloc().init()
            delegate._queue = self._queue
            app.setDelegate_(delegate)
            app.run()
    ```

    Avoid: calling any AppKit/NSPanel method from show()/hide() directly — always via queue.
    Avoid: `NSApplicationMain()` — needs Info.plist; use `app.run()` directly.

<task type="auto">
  <name>Task 3: Update __main__.py — overlay AppKit mainloop on main thread</name>
  <files>src/local_whisper/__main__.py</files>
  <action>
    Replace the `if args.run:` branch:

    ```python
    if args.run:
        from local_whisper.app import App
        from local_whisper.overlay import RecordingOverlay

        overlay = RecordingOverlay()
        app = App(overlay=overlay)
        app.start()  # starts pynput listener in daemon thread (non-blocking)

        try:
            overlay.run()  # AppKit event loop on main thread — blocks until quit()
        except KeyboardInterrupt:
            pass
        finally:
            app.stop()
            overlay.quit()
    ```

    Key: `app.start()` runs before `overlay.run()` so the listener is active before
    the AppKit event loop takes over the main thread. Ctrl+C raises KeyboardInterrupt
    which exits `overlay.run()`, then cleanup runs.

    Avoid: calling `App.run()` when overlay is active — it would block on threading.Event
    and never reach overlay.run().
