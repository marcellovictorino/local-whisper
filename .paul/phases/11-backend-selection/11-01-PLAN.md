<objective>
## Goal
Add `parakeet-mlx` as a faster inference backend (~0.3–0.5s vs ~0.5–0.7s for distil-whisper).
Backend is inferred automatically from the model ID — no separate config key. Users switch
backends by changing `[whisper] model` to a parakeet model ID. A `KnownModel` StrEnum lists
all supported models with their backend assignment, making extension trivial (add one entry).

## Purpose
Speed is the key differentiator. parakeet-mlx is pure Python/MLX — same install story as
mlx-whisper. No compilation, no Swift, no CoreML friction. Switching is invisible to users:
they set a model ID they want; the right backend loads automatically.

Using a StrEnum for known models provides:
- Single source of truth for supported model IDs
- Auto-inferred backend — no second config key to keep in sync
- Unknown model IDs fall back to mlx-whisper safely
- Adding future models = one line in KnownModel + one entry in _BACKEND_MAP

## Output
- `KnownModel` StrEnum listing all supported models with their HF IDs
- `_BACKEND_MAP: dict[str, str]` mapping model ID → backend name
- `get_backend(model: str) -> str` — pure lookup, no file I/O
- `run()` and `warm_up()` accept `backend` param, dispatch to backend implementation
- parakeet-mlx optional extra in pyproject.toml
- Benchmark updated to report backend alongside model
- 6+ new tests
</objective>

    **Add KnownModel + _BACKEND_MAP after DEFAULT_MODEL block:**
    ```python
    class KnownModel(StrEnum):
        """Supported model IDs with known backend assignments.

        Add new models here to register them. Unknown IDs fall back to mlx-whisper.
        """
        DISTIL_WHISPER = "mlx-community/distil-whisper-large-v3"
        WHISPER_TURBO  = "mlx-community/whisper-large-v3-turbo"
        PARAKEET_V2    = "mlx-community/parakeet-tdt-0.6b-v2"   # English
        PARAKEET_V3    = "mlx-community/parakeet-tdt-0.6b-v3"   # 25 European languages

    _BACKEND_MAP: dict[str, str] = {
        KnownModel.DISTIL_WHISPER: "mlx-whisper",
        KnownModel.WHISPER_TURBO:  "mlx-whisper",
        KnownModel.PARAKEET_V2:    "parakeet-mlx",
        KnownModel.PARAKEET_V3:    "parakeet-mlx",
    }

    DEFAULT_BACKEND = "mlx-whisper"


    def get_backend(model: str) -> str:
        """Infer backend from model ID. Unknown IDs default to mlx-whisper.

        Args:
            model: HuggingFace model ID (from get_model() or KnownModel).

        Returns:
            Backend name: "mlx-whisper" or "parakeet-mlx".
        """
        return _BACKEND_MAP.get(model, DEFAULT_BACKEND)
    ```

    **Add internal dispatch helpers (after get_backend):**
    ```python
    def _run_mlx_whisper(audio: np.ndarray, model: str) -> str:
        import mlx_whisper
        result = mlx_whisper.transcribe(audio, path_or_hf_repo=model, verbose=False)
        return result["text"].strip()


    def _run_parakeet(audio: np.ndarray, model: str) -> str:
        # Adjust API per Task 1 spike findings:
        try:
            import parakeet_mlx
        except ImportError:
            print(
                "[local-whisper] parakeet-mlx not installed. "
                "Run: uv sync --extra parakeet\nFalling back to mlx-whisper.",
                file=sys.stderr,
            )
            return _run_mlx_whisper(audio, model)
        parakeet_model = parakeet_mlx.from_pretrained(model)
        result = parakeet_model.transcribe(audio)
        return result["text"].strip()  # adjust per actual return type
    ```
