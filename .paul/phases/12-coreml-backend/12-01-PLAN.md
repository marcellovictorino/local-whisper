<objective>
## Goal
Add `whisperkittools` as a CoreML/ANE backend for local transcription. Wire it through the
existing KnownModel/backend dispatch infrastructure. Target: 200–500ms transcription latency
on M1/M2/M3 vs ~1s with mlx-whisper.

## Purpose
Phase 12 completes v0.6 Speed. CoreML/ANE is the fastest viable Python path — pre-built arm64
wheels (no Xcode), full ASR pipeline via `whisperkittools`, pre-compiled models from
`argmaxinc/whisperkit-coreml`. The existing multi-backend dispatch (Phase 11) means
integration is a single new KnownModel entry + one new `_run_*` function.

## Output
- `KnownModel.DISTIL_WHISPER_COREML` wired through dispatch
- `_run_whisperkittools()` in transcribe.py
- `warm_up()` pre-compiles ANE at startup (prevents 4-min delay at first keypress)
- `coreml` optional extra in pyproject.toml
- Tests cover new model + backend + ImportError fallback
- `benchmark_compare.py` includes coreml model for comparison
</objective>
