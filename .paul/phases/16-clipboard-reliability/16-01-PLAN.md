<objective>
## Goal
Add `settle_ms` and `retries` keyword arguments to `write_and_paste()`, making the paste contract explicit. Retry loop retries on `CalledProcessError`; settle delay fires before first attempt. Defaults keep current behavior unchanged.

## Purpose
The current `write_and_paste(text)` hides its reliability policy (no retry, no settle delay) inside the implementation. Callers cannot tune it without modifying the function. Making these params explicit at the interface level lets `app.py` express intent (e.g., settle after focus change) without owning the timing mechanism. This is the Clipboard Reliability Policy described in Phase 16 of the v0.8 roadmap.

## Output
- `clipboard.py`: `write_and_paste(text, *, settle_ms=0, retries=0)` with retry loop
- `tests/test_clipboard.py`: 3 new tests covering retry policy + settle delay
</objective>

    1. `pyperclip.copy(text)` — unchanged, always first
    2. `if settle_ms > 0: time.sleep(settle_ms / 1000)` — pre-paste delay
    3. Retry loop: `for attempt in range(1 + retries):` — attempt the osascript call
       - On success: `logger.info("Pasted %d chars.", len(text)); return`
       - On `CalledProcessError`: save exc as `last_exc`; if `attempt < retries`, `time.sleep(0.1)` then continue
    4. After loop exits (all attempts failed): extract stderr from `last_exc`, call `logger.warning(...)` mentioning attempt count

    Warning message template:
      `"Paste via osascript failed (after %d attempt(s)): %s — text copied to clipboard, paste manually with Cmd+V.", 1 + retries, stderr_msg`

    Do NOT add any other new logic. Do NOT change the osascript command string.
    Do NOT import anything beyond `time` (already have logging, subprocess, pyperclip).

        sleep_calls = []
        with (
            patch("pyperclip.copy"),
            patch("subprocess.run"),
            patch("time.sleep", side_effect=lambda s: sleep_calls.append(s)),

    ```python
    def test_retries_on_failure_success_on_second_attempt() -> None:
        call_count = []
        def _run_side_effect(*a, **kw):
            call_count.append(1)
            if len(call_count) == 1:
                raise subprocess.CalledProcessError(1, "osascript")
        with (
            patch("pyperclip.copy"),
            patch("subprocess.run", side_effect=_run_side_effect),
            patch("time.sleep"),
        ):
            write_and_paste("hello", retries=1)
        assert len(call_count) == 2
    ```

    Test 3 — all retries exhausted, warning logged, clipboard preserved:
    ```python
    def test_all_retries_exhausted_logs_warning_and_preserves_clipboard() -> None:
        copied = []
        with (
            patch("pyperclip.copy", side_effect=lambda t: copied.append(t)),
            patch("subprocess.run", side_effect=subprocess.CalledProcessError(1, "osascript")),
            patch("time.sleep"),
            patch("local_whisper.clipboard.logger") as mock_logger,
        ):
            write_and_paste("data", retries=1)
        assert copied == ["data"]
        mock_logger.warning.assert_called_once()
        call_args = mock_logger.warning.call_args[0]
        assert "2" in str(call_args) or 2 in call_args  # attempt count in warning
    ```
