<objective>
## Goal
Make local-whisper installable and auto-startable by a non-developer Mac user:
1. Accessibility permission check with actionable error on startup
2. `make install` sets up launchd user agent — app starts on login automatically
3. `make uninstall` cleanly removes it

## Purpose
Phases 1-3 produce a working tool that requires `uv run python -m local_whisper --run` every session. Distribution closes the gap: one install command, then it just works on every login. The Accessibility permission check prevents the silent failure that confused users in Phase 2 testing.

## Output
- `src/local_whisper/__main__.py` — Accessibility check on `--run`, clear error + instructions
- `setup.sh` — one-shot install script: uv sync + model download + launchd setup (git clone friendly, pipeable)
- `justfile` — day-to-day ops: start, stop, logs, uninstall, run (requires `just`)
</objective>

  <name>Task 1: Accessibility permission check in --run startup</name>
  <files>src/local_whisper/__main__.py</files>
  <action>
    Add `_check_accessibility()` function at the top of `main()` in the `if args.run:` branch, before importing App or RecordingOverlay.

    Use `ctypes` to call `AXIsProcessTrustedWithOptions` directly — no new dependency:

    ```python
    def _check_accessibility() -> bool:
        """Return True if this process has Accessibility permission."""
        import ctypes
        import ctypes.util
        lib_path = ctypes.util.find_library("ApplicationServices")
        if not lib_path:
            return True  # can't check, proceed optimistically
        try:
            lib = ctypes.cdll.LoadLibrary(lib_path)
            lib.AXIsProcessTrusted.restype = ctypes.c_bool
            lib.AXIsProcessTrusted.argtypes = []
            return bool(lib.AXIsProcessTrusted())
        except Exception:
            return True  # can't check, proceed optimistically
    ```

    In `if args.run:`, call it first:
    ```python
    if not _check_accessibility():
        print(
            "Accessibility permission required.\n"
            "\n"
            "  1. Open: System Settings → Privacy & Security → Accessibility\n"
            "  2. Add and enable the app running this script\n"
            "     (Terminal, iTerm2, or the launchd wrapper — whichever launched this)\n"
            "  3. Re-run: uv run python -m local_whisper --run\n",
            file=sys.stderr,
        )
        sys.exit(1)
    ```

    Place this check BEFORE any import of App, overlay, or pynput — those imports may
    partially initialise pynput's listener machinery. Early exit is cleaner.

    Avoid: importing ApplicationServices via pyobjc (not in pyproject.toml).
    Avoid: prompting user to open System Settings programmatically — just print the path.
  </action>
  <verify>
    # With permission granted (normal case):
    uv run python -m local_whisper --run &amp; sleep 1 &amp;&amp; kill %1
    # Should start normally (no accessibility error)

    # Can't automate the denial case — verified manually at checkpoint
  </verify>
  <done>AC-1 satisfied: clear error + sys.exit(1) when Accessibility not granted</done>

<task type="auto">
  <name>Task 2: setup.sh — one-shot install script</name>
  <files>setup.sh</files>
  <action>
    Create `setup.sh` in the project root. This is the primary install path:
    `git clone <repo> && cd local-whisper && bash setup.sh`

    Script structure (bash, strict mode: `set -euo pipefail`):

    **Step 1 — Check prerequisites:**
    - `uv` must be installed. If not: print install instructions (`curl -LsSf https://astral.sh/uv/install.sh | sh`) and exit 1.
    - macOS only check: `[[ "$(uname)" == "Darwin" ]]` or exit with message.

    **Step 2 — Install Python deps:**
    ```bash
    echo "Installing dependencies..."
    uv sync
    ```

    **Step 3 — Pre-download model:**
    ```bash
    echo "Checking model cache (may download ~1.5 GB on first run)..."
    uv run python -c "
    import numpy as np
    from local_whisper.transcribe import run
    run(np.zeros(1600, dtype='float32'))
    "
    echo "Model ready."
    ```

    **Step 4 — Write and load launchd plist:**
    Variables resolved at runtime:
    ```bash
    PROJECT_DIR="$(cd "$(dirname "${BASH_SOURCE[0]}")" && pwd)"
    UV_BIN="$(which uv)"
    PLIST_NAME="com.local-whisper"
    PLIST_DEST="$HOME/Library/LaunchAgents/$PLIST_NAME.plist"
    LOG_FILE="$HOME/Library/Logs/local-whisper.log"
    ```

    Write plist with heredoc:
    ```bash
    mkdir -p "$HOME/Library/LaunchAgents" "$HOME/Library/Logs"
    cat > "$PLIST_DEST" <<PLIST
    <?xml version="1.0" encoding="UTF-8"?>
    <!DOCTYPE plist PUBLIC "-//Apple//DTD PLIST 1.0//EN"
      "http://www.apple.com/DTDs/PropertyList-1.0.dtd">
    <plist version="1.0">
    <dict>
        <key>Label</key><string>$PLIST_NAME</string>
        <key>ProgramArguments</key>
        <array>
            <string>$UV_BIN</string>
            <string>run</string>
            <string>--project</string>
            <string>$PROJECT_DIR</string>
            <string>python</string>
            <string>-m</string>
            <string>local_whisper</string>
            <string>--run</string>
        </array>
        <key>RunAtLoad</key><true/>
        <key>WorkingDirectory</key><string>$PROJECT_DIR</string>
        <key>StandardOutPath</key><string>$LOG_FILE</string>
        <key>StandardErrorPath</key><string>$LOG_FILE</string>
        <key>EnvironmentVariables</key>
        <dict>
            <key>PYTHONWARNINGS</key>
            <string>ignore::UserWarning:multiprocessing</string>
        </dict>
    </dict>
    </plist>
    PLIST
    ```

    Load the service:
    ```bash
    launchctl unload "$PLIST_DEST" 2>/dev/null || true
    launchctl load -w "$PLIST_DEST"
    ```

    **Step 5 — Accessibility permission reminder:**
    Print clear instructions AFTER install (service has started, but may fail silently
    until permission granted):
    ```bash
    echo ""
    echo "✓ local-whisper installed. Starts automatically on login."
    echo ""
    echo "IMPORTANT: Grant Accessibility permission to complete setup:"
    echo "  System Settings → Privacy & Security → Accessibility"
    echo "  Add and enable the process running local-whisper"
    echo "  (Terminal, or the uv binary: $UV_BIN)"
    echo ""
    echo "Logs: $LOG_FILE"
    echo "To uninstall: just uninstall  (or: launchctl unload $PLIST_DEST && rm $PLIST_DEST)"
    ```

    Note: `PYTHONWARNINGS=ignore::UserWarning:multiprocessing` in plist suppresses
    leaked semaphore warning from pynput on exit.

    Note: No `KeepAlive` in plist — prevents restart loop if Accessibility not yet granted.

    Make executable: chmod +x setup.sh at the end of creation is not needed (user will
    call `bash setup.sh`), but add shebang `#!/usr/bin/env bash` for direct execution.

    Avoid: `curl | bash` pattern in the script itself — that's the user's invocation pattern,
    not something the script orchestrates.
    Avoid: sudo — all paths are under $HOME.
  </action>
  <verify>
    bash -n setup.sh  # syntax check
    bash setup.sh --help 2>&amp;1 || true  # shouldn't crash on unknown flag (set -e)
    # Real verification at human checkpoint
  </verify>
  <done>AC-2, AC-3, AC-5 satisfied: setup.sh installs service, pre-downloads model</done>

<task type="auto">
  <name>Task 3: justfile — day-to-day operational commands</name>
  <files>justfile</files>
  <action>
    Create `justfile` for users who have `just` installed. Complements setup.sh —
    setup.sh for install, justfile for ongoing operations.

    ```just
    project_dir := justfile_directory()
    uv          := `which uv`
    plist_name  := "com.local-whisper"
    plist_dest  := env_var("HOME") / "Library/LaunchAgents" / plist_name + ".plist"
    log_file    := env_var("HOME") / "Library/Logs/local-whisper.log"

    # Install local-whisper as a background service (starts on login)
    [group('setup')]
    install:
        bash {{justfile_directory()}}/setup.sh

    # Remove the background service
    [group('setup')]
    uninstall:
        launchctl unload {{plist_dest}} 2>/dev/null || true
        rm -f {{plist_dest}}
        @echo "local-whisper uninstalled."

    # Start the service (without reinstalling)
    [group('service')]
    start:
        launchctl start {{plist_name}}

    # Stop the service (without uninstalling)
    [group('service')]
    stop:
        launchctl stop {{plist_name}}

    # Show service status
    [group('service')]
    status:
        launchctl list | grep {{plist_name}} || echo "Not loaded"

    # Run in foreground (for debugging — Ctrl+C to quit)
    [group('dev')]
    run:
        {{uv}} run python -m local_whisper --run

    # Stream service logs
    [group('dev')]
    logs:
        tail -f {{log_file}}
    ```

    `install` delegates to `setup.sh` — single source of truth for install logic.

    Avoid: duplicating plist generation logic from setup.sh in justfile.
    Avoid: requiring `just` for the install path (that's what setup.sh is for).
  </action>
  <verify>
    just --list
    # Should list all public recipes organised by group
  </verify>
  <done>AC-4 satisfied: uninstall recipe; day-to-day ops available via just</done>
