## Goal
Set up the Python project and implement the full audio → text pipeline: capture microphone audio, transcribe with a local MLX model, print result. No hotkey or UI yet.

## Purpose
Validates the core value — local, offline transcription on Apple Silicon — before adding any interaction layer. If this works, everything else is integration.

## Output

- `python -m local_whisper --test` records 5 seconds, prints transcript
- `pyproject.toml` with all dependencies pinned

## Project Context
@.paul/PROJECT.md

## Tech Decisions
- **ASR framework:** `mlx-whisper` — runs Whisper models natively on Apple Silicon via MLX. Best on-device performance on M-family chips. Model: `mlx-community/whisper-large-v3-turbo` (fast, high accuracy).
- **Audio capture:** `sounddevice` — cross-platform, NumPy-native, no extra C libs needed.
- **Package manager:** `uv` — fast, modern, handles virtual envs automatically.
</context>

<task type="auto">
  <name>Task 2: Audio capture module</name>
  <files>src/local_whisper/audio.py</files>
  <action>
    Create `src/local_whisper/audio.py` with a single public function:

    ```python
    def record(duration: float, sample_rate: int = 16000) -> np.ndarray:
        """Record audio from default microphone.

        Args:
            duration: Recording length in seconds.
            sample_rate: Sample rate in Hz. Whisper expects 16000.

        Returns:
            Float32 numpy array of shape (N,) normalised to [-1.0, 1.0].
        """
    ```

    Implementation:
    - Use `sounddevice.rec(int(duration * sample_rate), samplerate=sample_rate, channels=1, dtype="float32")`
    - Call `sounddevice.wait()` to block until recording finishes
    - Return array squeezed to 1D (shape: (N,))
    - Print "Recording..." before and "Done." after to stderr

    Avoid: threading at this stage — blocking record is correct for Phase 1.
    Avoid: writing to disk — return numpy array directly.
  </action>
  <verify>uv run python -c "from local_whisper.audio import record; a = record(2); print(a.shape, a.dtype)" prints shape (32000,) float32</verify>
  <done>AC-2 satisfied: returns correct numpy array at 16kHz</done>
</task>
