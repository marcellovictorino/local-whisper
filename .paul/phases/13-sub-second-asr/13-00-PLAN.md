<objective>
## Goal
Spike all three sub-second ASR candidates on this machine; measure latency and WER for each viable path; recommend the winner for Phase 13 implementation.

## Purpose
Current parakeet-mlx delivers ~0.3–0.5s transcription. The v0.7 target is <1s keypress-to-paste for typical dictation (5–15s audio). Need evidence before committing to a new backend — each candidate has very different install friction, accuracy risk, and distribution complexity.

- `spikes/whispercpp_spike.py` — whisper.cpp CoreML spike script (only if SFSpeechRecognizer rejected)

</objective>
